# Development plans

This folder holds **in-repo snapshots** of substantial design or implementation plans (often authored in Cursor before coding).

- Use **kebab-case** filenames (e.g. `elevate-todo.md`).
- When a plan drives shipped work, link it from an entry in [changelog.md](../../changelog.md) under `## [Unreleased]` or the relevant version section.
- Omit plan links for drive-by fixes, dependency bumps, or work with no plan artifact.

Cursor may keep ephemeral copies under your local `.cursor/plans/`; only files committed here are shared with contributors.
