## Quick reference

- Admin nav: **Admin → Identity Providers** (not "System Config")
- Production requires `AUTH_SECRETS_KEY`, `BACKEND_PUBLIC_ORIGIN`, `FRONTEND_ORIGIN`
- Group→role mappings are promote-only
